# User Stories – GiftLink

## As a user, I want to:
1. Register for an account so that I can list or request items.
2. Log in securely so that my data is protected.
3. Browse all available items on the main page.
4. Search for items by category (e.g., furniture, electronics, clothes).
5. View detailed information about a specific item.
6. Add a comment to an item listing.
7. Edit my profile (name, email, password).
8. List an item I want to give away.
